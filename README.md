# React Redux Form Wizard

**Multi-step form wizard built with React and Redux.**  
Features controlled inputs, step navigation, clickable step indicator, and form validation.

---

## Features

- Multi-step form with 4 steps:
  1. Personal Info
  2. Address Info
  3. Account Details
  4. Review & Submit
- Step indicator showing progress and clickable steps
- Controlled inputs using Redux state
- Form validation: Submit disabled until required fields are filled
- Reset form functionality

---

## Tech Stack

- React.js  
- Redux Toolkit  
- React Redux  
- CSS / Tailwind (optional)  

---

```bash
